## Описание

Решение 3D задачи линейной упругости методом конечных элементов.  
Две реализации: **последовательная** (deal.II) и **параллельная** (MPI+PETSc).

## Задача

Куб [0,1]³, сталь: E = 2e11 Па, ν = 0.3.  
- Нижняя грань (x₃=0): закреплена (u=0)  
- Верхняя грань (x₃=1): нагрузка t₃ = 1e9·x₁ Па  
- Остальные грани свободны.  
Сетка 10×10×10, линейные элементы (Q1).

## Требования

- CMake ≥ 3.10
- deal.II ≥ 9.5 (с PETSc и MPI)
- PETSc
- MPI
- C++17

## Сборка

### Последовательная

```bash
cd sequential_elasticity
mkdir build && cd build
cmake ..
make
./elasticity_seq
```

### Параллельная

```bash
cd parallel_elasticity
mkdir build && cd build
cmake ..
make
mpirun -np 4 ./elasticity_par
```

## Ограничения

Для просмотра визуализации параллельной программы, в ParaView необходимо импортировать .vtu-файлы всех потоков.
